# HAVEN iPhone Production Suite

Open `index.html` in the workspace preview to view the full HAVEN mobile suite.

What is included:
- iPhone-sized elder app with exact-window fit.
- English default UI with a one-tap Dutch switch.
- High-contrast and font-size controls.
- SCHILD, ANKER, KRING, BUURT, KOMPAS, STEM and WACHT screens.
- Local service-layer state for medications, scam events, document scan summaries, family messages, life stories, BUURT matching, care logs and notifications.
- Family dashboard and care portal mode inside the same mobile shell.

Use the home screen feature cards to reach STEM, KOMPAS, WACHT and settings. Use the bottom navigation for the daily elder flows.
