# HAVEN Support Model

## Support channels

- Product support: support@haven.nl
- Privacy requests: privacy@haven.nl
- Security reports: security@haven.nl

## Severity levels

| Severity | Description | Response target |
|---|---|---:|
| P0 | Data exposure, production outage, crisis flow failure | 1 hour |
| P1 | Medication, scam, location or emergency feature degraded | 4 hours |
| P2 | Family dashboard, care portal or non-critical feature issue | 1 business day |
| P3 | Documentation, cosmetic or minor issue | next planned release |

## User support principles

- Use calm language.
- Never shame the elder.
- Confirm consent before discussing another person's data.
- Escalate privacy/security concerns immediately.
