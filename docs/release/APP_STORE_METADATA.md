# HAVEN App Store Metadata

## App name
HAVEN — Elder Companion

## Subtitle
Safety, medicine, family and dignity

## Description
HAVEN is a voice-first elder-care companion for older adults and the families who support them. It combines calm medication reminders, family messages, scam protection, safe-zone support, life stories and emergency profile access in one dignified app.

HAVEN is not a surveillance app. Location is privacy-preserving, consent-based and fuzzed for family views. The elder controls sharing and can revoke consent.

## Keywords
elder care, medication reminder, family care, scam protection, senior safety, caregiver, wellbeing

## Privacy summary
HAVEN processes profile, medication, family message, voice interaction, location event and wellbeing data only for the features enabled by the elder. HAVEN does not collect BSN and warns users not to upload documents containing BSN.
